Java মানেই কী Android ?

JAVA নামটি শুনলেই আমরা একটি কথাই মনে করি আমি Android নিয়ে কাজ করবো । Android নিয়ে কাজ করার জন্য জাভা প্রয়োজন কিন্তু জাভা মানেই Android এইটা একদম ভুল ধারনা । JAVA এমন একটা Language যা দিয়ে প্রোগ্রামিং জগতে করা যায় না এমন কাজ নেই বললেই চলে । Android মার্কেটে আসে ২০০৫ সালে এবং ২০০৭ সালে Google তা Develop করে । JAVA প্রথম আরম্ভ করে ১৯৯১ সালের June মাসে যার নাম ছিলো ' Oak ' । ১৯৯৫ সালের দিকে নাম পরির্তন করে JAVA রাখা হয় । আমি এই সব নিয়ে আলোচনা করবো না । যদি জানা প্রয়োজন মনে হয় Wikipedia থেকে দেখে নিবেন। JAVA নিয়ে বিস্তারিত আলোচনা করা আমার মূল উদ্দেশ্য নয় , মূল উদ্দেশ্য হল ১৯৯১ সাল থেকে ২০০৫ অথবা ১৯৯৫ সাল থেকে ২০০৫ সাল পর্যন্ত যে (১০ - ১৪) বছর JAVA নিয়ে Programmer কাজ করতো , তারা আসলে কী কাজ করেছে? Android যদি JAVA মূল কাজ হয় তা হলে একটা প্রশ্ন থেকেই যায় (১৯৯১ - ২০০৫) বা (১৯৯৫ - ২০০৫) এই (১০ - ১৪) বছর JAVA প্রোগ্রামারেরা কী কাজ করত ? আমি প্রথমে বলেছি JAVA এমন একটা Programming Language যা দিয়ে করা যায় না এমন কাজ নেই । Android আসার আগে JAVA দিয়ে Web Programming এর কাজ করা হতো । বিশ্বের জনপ্রিয় যত সাইড আছে তার মধ্যে প্রায় সব গুলোতে JAVA Applet ব্যবহার করা হয়েছে । YouTube, LinkedIn, Google Search, Amazon, Twitter এমন কি Microsoft ও JAVA ব্যবহার করে তৈরি করা হয়েছে। এবার একটু বিশ্বের মধ্যে জনপ্রিয় কিছু বিষয় নিয়ে বলি....... যেমনঃ Robotics । Robotics এ যে সব Language নিয়ে কাজ করে তা হলো : Prolog , Python , Lisp তার মধ্যেও JAVA এর ব্যবহার প্রয়োজন । বর্তমান বিশ্বে সব চাইতে আলোচিত একটা বিষয় Data Mining এবং Data Science । সবাই এখন এই বিষয়টির দিকে বেশি জোর দিচ্ছে Data Mining নিয়ে কাজ করার জন্য R, Python, SQL, SAS, MATLAB এই Language এর সাথে JAVA অত্যন্ত প্রয়োজনীয় একটা Language । JAVA নিয়ে এই কথাগুলো বলার কারন হলো..... প্রায় JAVA নিয়ে যারা কাজ করা শুরু করে সবাই Basic Java Idea না নিয়ে Android এ কাজ করা শুরু করে এর জন্য অনেকেই Android Platform এ কিছু করতে অনেক সমস্যায় পড়ে । আবার অনেকে হাল ছেড়েও দেয় । আমার মতে Android এ কাজ করার আগে Java basic Concept এবং Java দিয়ে কি কাজ করা যায় তার ধারনা রাখা দরকার । আজকে আর নয় । অন্য একদিন Java Socket আর Hacking এ JAVA এর ব্যবহার নিয়ে আলোচনা করব ।
